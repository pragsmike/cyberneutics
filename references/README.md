# References

This directory contains background reading and theoretical foundations for the Cyber-Sense methodology.

## Planned Contents

*   **Dervin's Sense-Making Methodology**: Resources on Brenda Dervin's situation-gap-bridge model.
*   **Second-Order Cybernetics**: Works by Heinz von Foerster, Gregory Bateson, and others on the role of the observer in systems.
*   **Deleuzian Philosophy**: Concepts of becoming, difference, and repetition relevant to narrative computing.
*   **[Narrative Engineering](https://alexbo.land/essays/narrative_engineering_2023.html)** (Alex Boland): Foundational essay on steering narrative potential using diegetic frames.
*   **Life Itself** (Robert Rosen): Comprehensive proof that mechanism (numeric/symbolic) cannot simulate living systems (narrative/semantic), necessitating a different category of modeling for complex systems.
